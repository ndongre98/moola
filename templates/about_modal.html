About moola

Moola is a stock sentiment analysis tool. For each company that is queried, moola scrapes relevant articles from Motley Fool and then uses IBM Watson's NLP API to perform a sentiment reading on these articles.

Moola gives users an emotional analysis reading for each company by providing a score from 0 to 1 on 5 emotional axes:

  1. Sadness
  2. Joy
  3. Fear
  4. Disgust
  5. Anger

Moola also provides a sentiment analysis score:

  • Positive: Scores between 0.34 and 1.00
  • Neutral: Scores between -0.33 and 0.33
  • Negative: Scores between -1.0 and 0.32
